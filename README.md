# CallingApp — Production WhatsApp Clone (Capacitor + Render + Cloudinary)

> Native FCM push, WebRTC voice/video calls, **media messages (image / video / audio / file / voice notes)** via **Cloudinary signed uploads**, online presence, typing indicators, read receipts, message reactions, reply-to and profile photos.

```
mobile-app/                     # Capacitor wrapper (Android/iOS) — wraps the web app + native FCM
  www/index.html                # Main UI (HTML messenger)
  www/fcm.js                    # Native FCM init (foreground + background)
  www/cloudinary.js             # MediaUploader — signed direct uploads to Cloudinary
  www/whatsapp-features.js      # WhatsApp-clone feature pack (media, voice, presence, ...)
notification-server/            # Express + Firebase Admin + Cloudinary helper (Render-ready)
  index.js                      # All routes — FCM + Cloudinary signing
  cloudinary.js                 # Signing / destroy helpers
  .env.example                  # Sample env (incl. Cloudinary)
  render.yaml                   # Render Blueprint
.github/workflows/              # GitHub Actions — auto-build Android APK
```

## 5. API reference

### FCM endpoints  (existing — unchanged contract)

| Method | Path | Auth | Purpose |
|--------|------|------|---------|
| POST   | `/api/register-token`   | —    | Register a device token |
| POST   | `/api/send`             | key  | Send to user (multi-device fan-out) |
| POST   | `/api/send-to-token`    | key  | Send to a raw FCM token |
| POST   | `/api/send-call-cancel` | key  | Dismiss ringing |
| POST   | `/api/send-missed-call` | key  | Missed-call alert |
| POST   | `/api/send-multi`       | key  | Up to 500 recipients |
| POST   | `/api/schedule`         | key  | Future delivery |
| POST   | `/api/validate-token`   | key  | Dry-run validity |
| POST   | `/api/delivery-receipt` | key  | Client ACK / analytics |
| GET    | `/api/stats`            | key  | Counters |
| GET    | `/api/metrics`          | key  | Prometheus text |

### Cloudinary endpoints (new)

| Method | Path | Auth | Purpose |
|--------|------|------|---------|
| POST   | `/api/cloudinary/sign`    | **Bearer ID-token** | Sign params for client direct upload |
| POST   | `/api/cloudinary/destroy` | server key | Delete an asset |

`/api/cloudinary/sign` body: `{ chatId: string, kind: "image"|"video"|"audio"|"voice"|"file" }`
Headers: `Authorization: Bearer <Firebase ID token>`
Returns signed `{ cloud_name, api_key, timestamp, signature, folder, public_id, tags, type, resource_type, upload_url, max_bytes }`.

### Client usage (in the app)

```js
// Image
const res = await window.MediaUploader.upload(file, {
  chatId: "abc_def",
  kind: "image",
  onProgress: (pct) => console.log(pct + "%"),
});
console.log(res.url);          // permanent Cloudinary URL
console.log(res.publicId);     // for deletion

// Voice note (Blob from MediaRecorder)
const v = await window.MediaUploader.uploadBlob(blob, "note.webm", { chatId, kind: "voice" });

// Helpers
MediaUploader.thumb(url, 300);        // square thumb
MediaUploader.optimized(url, 1080);   // c_limit auto format/quality
MediaUploader.videoPoster(url, 600);  // first-frame JPG
```

## 8. Troubleshooting

- **`cloudinary_not_configured`** → Render env vars missing. Re-check `CLOUDINARY_*`.
- **`upload_failed_401`** → Firebase ID token expired; user needs to re-login.
- **Voice note button does nothing** → Mic permission denied; settings → app → permissions.
- **Render free plan cold start** → first call slow; upgrade plan or use cron-ping.
- **iOS** → `npx cap add ios`, upload APNs key to Firebase, add `GoogleService-Info.plist`. Mac required.

## 9. Security checklist

- ✅ `CLOUDINARY_API_SECRET` lives **only** on the server.
- ✅ All upload signing requires a valid Firebase ID token.
- ✅ Server rate-limits both `/api/send` and `/api/cloudinary/sign` per user.
- ✅ HSTS, helmet, CORS, request IDs, structured JSON logs.
- 🔒 Add Firebase RTDB security rules in production:

```jsonc
{
  "rules": {
    "messages":   { "$chatId": { ".read": "auth != null && $chatId.contains(auth.uid)",
                                 ".write": "auth != null && $chatId.contains(auth.uid)" } },
    "presence":   { "$uid": { ".read": "auth != null", ".write": "auth.uid === $uid" } },
    "fcmTokens":  { "$uid": { ".read": "auth.uid === $uid", ".write": "auth.uid === $uid" } },
    "users":      { "$uid": { ".read": "auth != null", ".write": "auth.uid === $uid" } },
    "typing":     { "$chatId": { ".read": "auth != null", ".write": "auth != null" } }
  }
}
```
